# 5e Combat Companion

A single-file D&D 5e combat tracker for experienced players. No install, no server, no account — just open `tracker.html` in any browser.

## Features
- HP tracking with color-reactive bar (green → yellow → red)
- Ability scores, saves, AC, speed, initiative
- Proficiency Bonus field
- Auto-calculated Spell Save DC and Spell Attack (with manual override)
- Spellcasting Ability selector (INT / WIS / CHA)
- Spell slots (all 9 levels, toggleable for martial characters)
- Active spells tracker with Concentration handling
- Conditions, Immunities & Resistances
- Optional Senses panel (Truesight, Blindsight, Tremorsense)
- Custom Attacks (name, to hit, damage, notes)
- Inventory and Magic Items (Equipped/Attuned tracking)
- Attuned Items tracker with expandable slot cap (house rule friendly)
- Formula reference dropdown in header
- Floating Stat Block Viewer (drag, pin, resize, drop images/PDFs)
- Resizable column layout
- All data persists in browser localStorage

## Usage
Download `tracker.html` and open it in your browser. Everything saves automatically.

## Class Variants
Class-specific variants (Druid with Wild Shape, etc.) are maintained on separate branches.
